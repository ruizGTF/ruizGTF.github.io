¿Que es CSS?

El CSS (Cascading Style Sheets) es un lenguaje de diseño gráfico utilizado para estilizar elementos escritos en un lenguaje de marcado, como HTML, SVG o XML.

¿Que aprendere en este curso?

En el curso de CSS aprenderás los fundamentos, sintaxis y buenas prácticas de este lenguaje de estilos Estos cursos están diseñados para llevar a los estudiantes desde un nivel principiante hasta un nivel experto en CSS de manera simple, rápida y efectiva. Al final del curso, los estudiantes deberían ser capaces de construir sitios web atractivos que se vean bien en cualquier dispositivo.
